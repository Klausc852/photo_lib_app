### Prerequisites
- Flutter SDK 3.0.0 or higher
- Dart SDK 3.0.0 or higher

### Installation

1. **Clone or copy this project structure**

2. **Install dependencies:**
```bash
cd Photo_Lib/
flutter pub get
```

3. **Run build_runner (for Hive):**
```bash
flutter pub run build_runner build
```

4. **Run the app:**
```bash
flutter run
```
